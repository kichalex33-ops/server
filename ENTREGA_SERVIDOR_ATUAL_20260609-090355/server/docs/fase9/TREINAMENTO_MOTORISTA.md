# Treinamento do Motorista

## Roteiro

- Entrar no app motorista.
- Conferir viagens atribuidas.
- Enviar checklist.
- Iniciar fluxo da viagem.
- Enviar GPS e eventos.
- Registrar ocorrencia.
- Acionar panico somente em emergencia real ou treinamento controlado.

## Validacao

O app deve enviar GPS e evento que aparecam no servidor.
