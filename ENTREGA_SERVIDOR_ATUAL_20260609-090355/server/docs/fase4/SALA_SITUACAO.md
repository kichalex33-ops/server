# Sala de Situacao

## Rota

`/painel-logistico/sala-situacao`

## Layout

- Mapa operacional ocupando aproximadamente 70% da area principal.
- Painel de comando ocupando aproximadamente 30%.
- Indicadores no topo.
- Lista de viagens ativas.
- Lista de alertas.
- Feed operacional.
- Alternancia real/demo.

## Operacao

O modo real consome `/api/live-map`. O modo demo usa dados locais do JavaScript para treinamento e fallback visual.

## Identidade

A tela segue identidade clara Andrade Gestao em Saude, com fundo claro, bordas discretas, dourado institucional e navegacao simples para voltar ao painel.
