# Piloto Real - Fase 7

## Objetivo

Validar a comunicacao entre servidor Linux real, plataforma Node e celulares Android reais.

## Ambiente recomendado

- Servidor Node em Linux.
- Celular Android com app Flutter instalado.
- Teste em Wi-Fi local.
- Teste em 3G, 4G e 5G quando disponivel.

## Acesso externo

Fora da rede local, o app precisara de um endereco HTTPS publicado por Cloudflare Tunnel, Ngrok, dominio com proxy reverso ou redirecionamento de porta seguro.

## Viagem piloto

Cenario minimo:

- 1 motorista
- 1 veiculo
- 1 viagem
- 2 pacientes
- 2 acompanhantes

Validar:

- login
- viagens atribuidas
- checklist
- KM inicial
- saida
- GPS
- embarque
- espera
- retorno
- finalizacao
- pânico
- comprovante
