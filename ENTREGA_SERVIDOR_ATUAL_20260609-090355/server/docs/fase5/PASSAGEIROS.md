# Passageiros

## Endpoints

- `GET /api/gestao/passageiros`
- `GET /api/gestao/absenteismo`

## Dados

- passageiros previstos;
- pacientes transportados;
- acompanhantes transportados;
- pacientes ausentes;
- desistencias;
- necessidades especiais;
- destinos frequentes;
- taxa de comparecimento.
