# Custos

## Endpoints

- `GET /api/gestao/custos`
- `GET /api/gestao/combustivel`

## Calculos

Os custos usam a collection `despesas`.

Campos calculados:

- custo total;
- combustivel;
- pedagio;
- estacionamento;
- manutencao;
- alimentacao;
- custo por km;
- custo por viagem;
- custo por paciente.
