# Timeline Operacional - Fase 3

## Endpoint

- `GET /api/viagens/:id/timeline`

## Fontes

A timeline combina:

- eventos;
- ocorrencias;
- mensagens;
- alertas;
- checklists;
- despesas.

Tudo e ordenado por data/hora.

## Exemplos de itens

- Preparacao iniciada.
- Saida confirmada.
- Passageiro embarcado.
- Passageiro ausente.
- Ocorrencia registrada.
- Despesa registrada.
- Viagem concluida.
