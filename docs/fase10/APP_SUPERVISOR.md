# App Supervisor

## Objetivo futuro

Aplicativo leve para gestor/supervisor acompanhar a operacao fora do painel web.

## Funcoes previstas

- Viagens do dia.
- Veiculos no mapa.
- Alertas criticos.
- Aprovar ou cancelar ocorrencias.
- Motoristas ativos.
- Painel resumido.
- Emergencias.
- Historico basico.

## Endpoints candidatos

- `GET /api/indicadores/gestor`
- `GET /api/live-map`
- `GET /api/emergencias`
- `GET /api/alertas`
- `GET /api/gestao/dashboard`
- `GET /api/auditoria`

## Decisao

Nao implementar app supervisor agora. Primeiro estabilizar piloto e autenticacao.
