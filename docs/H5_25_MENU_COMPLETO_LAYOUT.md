# H5.25 - Menu completo e layout de navegação

- Operador com menu completo e novas páginas de apoio.
- Gestor com menu no mesmo padrão visual do operador.
- Menu lateral com rolagem real da lateral inteira.
- Removido overlay escuro que deixava o painel apagado atrás do menu.
- Ícones e textos do menu forçados como visíveis.
- Cada item do Gestor aponta para página própria.
