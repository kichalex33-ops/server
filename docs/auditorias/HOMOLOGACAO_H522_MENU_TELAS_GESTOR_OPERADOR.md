# H5.22 - Menu por telas para Gestor e Operador

- Gestor passou a seguir o mesmo padrão visual do Operador.
- Cada item do menu do gestor abre uma página própria.
- As opções finais do gestor agora levam a telas reais: Operadores, Perfis e Parâmetros.
- Operador ganhou opções adicionais no menu: Frota de Veículos e Sincronização.
- Ícones SVG do menu receberam tamanho fixo para não sumirem.
- O ZIP mantém a pasta raiz homologacao/.
