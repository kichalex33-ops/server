# H5.23 - Correção do menu do gestor e operador

Correção direta para o menu lateral:

- Recria o menu do gestor via JS se ele estiver vazio ou quebrado.
- Recria o menu do operador com todas as opções necessárias.
- Força visibilidade de textos, ícones SVG e seções do menu.
- Mantém o menu sob demanda: abre pelos três tracinhos, fecha clicando fora ou com Esc.
- Mantém cada item do gestor apontando para uma página própria.

- `public/assets/css/h523-menu-fix.css`
- `public/assets/js/h523-menu-fix.js`
