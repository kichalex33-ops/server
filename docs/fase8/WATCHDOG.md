# Watchdog Avancado - Fase 8

## Endpoint

```text
GET /api/watchdog
```

## Monitores

- GPS parado
- Veiculo parado
- Espera excessiva
- Retorno nao iniciado
- Viagem sem atualizacao
- Sincronizacao travada
- Velocidade acima do limite

## Resultado

O endpoint retorna alertas gerados e lista dos monitores executados.
