# Treinamento do Operador

## Roteiro

- Acessar `/painel-logistico/operador`.
- Acompanhar viagens do dia.
- Usar sala de situacao para mapa e GPS.
- Verificar alertas e emergencias.
- Registrar mensagens e ocorrencias.
- Acionar protocolo em `/painel-logistico/emergencias`.

## Validacao

Operador deve simular uma viagem, um alerta, uma emergencia e uma finalizacao.
