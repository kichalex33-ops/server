# Treinamento do Gestor

## Roteiro

- Acessar `/painel-logistico/gestao`.
- Interpretar KPIs.
- Conferir rankings.
- Validar custos.
- Exportar CSV.
- Acompanhar auditoria.
- Consultar infraestrutura em `/painel-logistico/admin/infra`.

## Validacao

Gestor deve explicar indicadores, identificar alertas abertos e confirmar ultimo backup.
